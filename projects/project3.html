Project Overview
This project involved the management, validation, and maintenance of telecommunications infrastructure data within a GIS environment. The work focused on mapping and maintaining fiber optic network assets, including poles, access points, network equipment, cabinets, service boundaries, and fiber cable routes. Using GIS tools, network components were spatially validated, updated, and organized to support network planning, construction, quality control, and operational decision-making. Using a third-party GIS platform, scanned land use maps were georeferenced to their correct geographic location and digitized according to the official land use classifications provided in the map legend.

Objectives

⚪ Network expansion.
⚪ Fiber deployment planning.
⚪ Infrastructure maintenance.
⚪ Service coverage analysis.
⚪ Construction management.
