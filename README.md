# Featured Projects

## Real-Time Delivery Area (RTA) and Branch Assignment System

### Overview

A GIS-based delivery management system designed to determine the appropriate service branch and estimated delivery time based on customer location.

The system utilizes spatial queries, geocoding, and predefined delivery zones to automatically identify which branch serves a customer and estimate delivery coverage.

### Workflow

Customer Address
↓
Geocode Location
↓
Spatial Query
↓
Intersect Delivery Zone
↓
Find Assigned Branch
↓
Return ETA

### Technologies

- PostgreSQL/PostGIS

- Spatial Analysis
